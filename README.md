# 微信爬虫 Wechat-Crawler

功能:获取所有好友的性别，地理位置（省，市），并进行统计。获取所有好友的个人签名，并制作成词云。  


## Update（2018/10/24）:

### 新增获取好友头像功能：get_avatar_of_friends.py  


## 使用方法(设备需已安装python)：


下载本项目到本地，打开项目主目录，打开命令行，输入：

pip install -r requirements.txt

执行对应的py文件，例如：

python Wechat_Crawler.py

或者

python get_avatar_of_friends.py
